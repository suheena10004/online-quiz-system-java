
#  Java Quiz App

##  Project Description
This is a console-based Quiz Application developed using Java.  
It allows users to answer multiple-choice questions and displays the final score.

---

## Problem Statement
Traditional quiz systems require manual evaluation.  
This project automates quiz conduction and evaluation using Java.

---

##  Objectives
- Build an interactive quiz system
- Implement Java concepts like OOP and collections
- Provide instant feedback to users

---

##  Features
- Multiple choice questions
- Score calculation
- Interactive menu
- Instant result display

---

##  Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class
