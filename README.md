# online-quiz-system-java
Java CLI Quiz System
This is a console-based quiz application developed using Java.  
It allows users to answer multiple-choice questions and calculates their score at the end.

---
- Multiple-choice questions
- User input handling
- Instant feedback (Correct/Wrong)
- Final score display

---

- Java
- OOP (Object-Oriented Programming)
- Scanner Class

---


1. Clone the repository:
   git clone https://github.com/suheena10004/online-quiz-system-java.git

2. Go to project folder:
   cd online-quiz-system-java

3. Compile:
   javac Main.java

4. Run:
   java Main
